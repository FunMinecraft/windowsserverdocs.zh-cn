# <a name="appendix-l-events-to-monitor"></a>附录 l：事件到监视器

>适用于：Windows Server 2016，Windows Server 2012 R2、Windows Server 2012

  
## <a name="appendix-l-events-to-monitor"></a>附录 l：事件到监视器  
下表列出了应监视根据中提供的建议，您的环境中的事件[监视 Active Directory 中是否的危害的迹象](../../ad-ds/plan/security-best-practices/Monitoring-Active-Directory-for-Signs-of-Compromise.md)。 下表中，在"当前 Windows 事件 ID"列列出事件 ID 因为它目前在主要支持的版本的 Windows 和 Windows Server 中实现。  
  
列出了"旧版 Windows 事件 ID"列中的 Windows 的旧版本，如运行 Windows XP 客户端计算机的相应事件 ID 或之前和运行 Windows Server 2003 或更早版本的服务器。 "潜在危险"列标识是否该事件应该考虑的低中, 或高危险检测攻击，以及"事件摘要"列中的提供的简短描述事件。  
  
高潜在危险表示某个事件应进行调查。 中或低潜在危险意味着，如果他们出现意外或编号会大大测量内超过预期的基准仅调查这些事件。 创建要求强制调查响应警报之前，所有组织应在他们的环境都测试这些建议。 每个环境不同，并且的某些事件排名与高潜在危险有可能由其他无害事件。
