# <a name="bitsadmin-peers-and-clear"></a>bitsadmin 对等方，然后清除



清除对等端列表。

## <a name="syntax"></a>语法

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>示例

以下示例清除对等端列表。
```
C:\>bitsadmin /Peers /Clear
```
