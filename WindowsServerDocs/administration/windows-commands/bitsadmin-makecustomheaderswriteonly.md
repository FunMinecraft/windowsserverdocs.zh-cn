# <a name="bitsadmin-makecustomheaderswriteonly"></a>bitsadmin makecustomheaderswriteonly

生成作业的自定义 HTTP 标头的只写 （无法撤消）。

## <a name="syntax"></a>语法

```
bitsadmin /MakeCustomHeadersWriteOnly <Job>
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|作业|该作业的显示名称或 GUID|
