# <a name="bitsadmin-peers-and-discover"></a>bitsadmin 对等互连和发现



再次发现对等方。

## <a name="syntax"></a>语法

```
bitsadmin /Peers /Discover 
```

## <a name="BKMK_examples"></a>示例

下面的示例将发现对等方。
```
C:\>bitsadmin /Peers /Discover
```
