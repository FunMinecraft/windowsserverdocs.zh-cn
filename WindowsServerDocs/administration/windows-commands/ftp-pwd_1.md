# <a name="ftp-pwd_1"></a>ftp： pwd_1

>适用于：Windows Server （半年频道），Windows Server 2016，Windows Server 2012 R2，Windows Server 2012

显示当前远程计算机目录。   
## <a name="syntax"></a>语法  
```  
pwd  
```  
### <a name="parameters"></a>Parameters  
无  
## <a name="BKMK_Examples"></a>示例  
显示当前远程计算机目录。  
```  
pwd  
```
