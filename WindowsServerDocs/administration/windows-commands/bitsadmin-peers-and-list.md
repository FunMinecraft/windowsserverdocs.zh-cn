# <a name="bitsadmin-peers-and-list"></a>bitsadmin 同行和列表



列出了所有对等方。

## <a name="syntax"></a>语法

```
bitsadmin /Peers /List 
```

## <a name="BKMK_examples"></a>示例

下面的示例列出了所有对等方。
```
C:\>bitsadmin /Peers /List
```
