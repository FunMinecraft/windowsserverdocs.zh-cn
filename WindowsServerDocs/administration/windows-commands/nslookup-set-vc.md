# <a name="nslookup-set-vc"></a>nslookup set vc



指定在向服务器发送请求时使用或不使用虚拟线路。

## <a name="syntax"></a>语法

```
set [no]vc
```

## <a name="parameters"></a>Parameters

| 参数 |                                              描述                                               |
|-----------|--------------------------------------------------------------------------------------------------------|
| **novc**  | 指定在向服务器发送请求时从不使用虚拟线路。 默认值为**novc**。 |
|  **vc**   |             指定在向服务器发送请求时始终使用虚拟线路。             |
|   {帮助   |                                                   ?}                                                   |
