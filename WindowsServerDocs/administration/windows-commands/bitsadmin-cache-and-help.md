# <a name="bitsadmin-cache-and-help"></a>bitsadmin 缓存和帮助



显示**缓存**开关的命令行用法。

## <a name="syntax"></a>语法

```
Bitsadmin /Cache /Help 
```

## <a name="BKMK_examples"></a>示例

以下命令显示**缓存**开关的命令行帮助。
```
Bitsadmin /Cache /Help
```
