# <a name="set-verbose"></a>设置详细



指定是否在卷影副本创建过程提供详细输出。 如果使用不带参数，**设置详细**在命令提示符下显示的帮助。

## <a name="syntax"></a>语法

```
set verbose {on | off}
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|{上 | 关闭}|打开或关闭详细模式。|

## <a name="remarks"></a>备注

-   如果详细模式位于**设置**提供的编写器确定包含或排除的详细信息和元数据压缩和提取的详细信息。
