# <a name="ftp-cd"></a>ftp: cd

>适用于：Windows 服务器 （半年频道），Windows Server 2016 中，Windows Server 2012 R2、 Windows Server 2012

更改远程计算机上的工作目录。   
## <a name="syntax"></a>语法  
```  
cd <remotedirectory>  
```  
### <a name="parameters"></a>Parameters  

|     参数     |                                 描述                                 |
|-------------------|-----------------------------------------------------------------------------|
| <remotedirectory> | 你想要更改远程计算机上指定的目录。 |

## <a name="BKMK_Examples"></a>示例  
更改到在远程计算机上的目录**Docs**。  
```  
cd Docs  
```  
更改到在远程计算机上的目录**可能视频**。  
```  
cd  May Videos  
```
