# <a name="nslookup-set-all"></a>nslookup set all



打印的配置设置的当前值。

## <a name="syntax"></a>语法

```
set all 
```

## <a name="parameters"></a>Parameters

| 参数 | 描述 |
|-----------|-------------|
|   {帮助   |     ?}      |

## <a name="remarks"></a>备注

-   **设置所有**还将打印有关的默认服务器和计算机 （主机） 的信息。
