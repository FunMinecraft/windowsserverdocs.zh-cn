# <a name="delete"></a>“删除”



删除一个分区或卷。 它还从磁盘列表删除动态磁盘。

## <a name="syntax"></a>语法

```
delete disk
delete partition
delete volume
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|[删除磁盘](delete-disk.md)|从磁盘列表删除丢失的动态磁盘。|
|[删除分区](delete-partition.md)|删除分区。|
|[删除卷](delete-volume.md)|删除卷。|
