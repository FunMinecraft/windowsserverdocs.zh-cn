# <a name="delete"></a>delete



删除分区或卷。 它还从磁盘列表中删除动态磁盘。

## <a name="syntax"></a>语法

```
delete disk
delete partition
delete volume
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|[删除磁盘](delete-disk.md)|从磁盘列表中删除丢失的动态磁盘。|
|[删除分区](delete-partition.md)|删除分区。|
|[删除卷](delete-volume.md)|删除卷。|
