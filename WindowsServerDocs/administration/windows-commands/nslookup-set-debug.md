# <a name="nslookup-set-debug"></a>nslookup set debug



打开或关闭调试模式。

## <a name="syntax"></a>语法

```
set [no]debug
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|**nodebug**|关闭调试模式。 默认语法**nodebug**。|
|**debug**|启用调试模式。|
|**{help | ?}**|显示的短摘要**nslookup**子命令。|

## <a name="remarks"></a>备注

-   开启调试模式下，打印有关发送到服务器，并得到答案的数据包详细信息。
