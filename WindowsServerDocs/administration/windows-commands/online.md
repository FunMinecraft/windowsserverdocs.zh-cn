# <a name="online"></a>在线



进入联机状态的脱机磁盘或卷。

## <a name="syntax"></a>语法

```
Offline disk
Offline volume
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|[脱机磁盘](offline-disk.md)|将选中的脱机磁盘带到联机状态。|
|[脱机卷](offline-volume.md)|选中的脱机卷进入联机状态。|
