# <a name="end-backup"></a>结束备份




结束完整备份会话和问题**BackupComplete**具有合适的编写器状态，如果所需的事件。

## <a name="syntax"></a>语法

```
end backup
```

## <a name="remarks"></a>备注

-   为所有完整备份，应使用此命令。
