# <a name="end-backup"></a>结束备份




结束完整备份会话，并根据需要发出具有相应编写器状态的**BackupComplete**事件。

## <a name="syntax"></a>语法

```
end backup
```

## <a name="remarks"></a>备注

-   应将此命令用于所有完整备份。
