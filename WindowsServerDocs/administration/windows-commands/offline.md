# <a name="offline"></a>脱机



联机磁盘或卷进入脱机状态。

## <a name="syntax"></a>语法

```
Offline disk
Offline volume
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|[脱机磁盘](offline-disk.md)|将具有焦点的联机磁盘带到脱机状态。|
|[脱机卷](offline-volume.md)|具有焦点的联机卷进入脱机状态。|
