# <a name="bitsadmin-cache-and-list"></a>bitsadmin 缓存和列表



列出所有缓存条目。

## <a name="syntax"></a>语法

```
bitsadmin /Cache /List [/Verbose] 
```

## <a name="BKMK_examples"></a>示例

下面的示例列出了详细格式的所有缓存条目。
```
C:\> bitsadmin /Cache /List /Verbose
```
