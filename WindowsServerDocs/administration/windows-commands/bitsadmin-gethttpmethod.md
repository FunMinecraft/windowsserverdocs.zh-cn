# <a name="bitsadmin-gethttpmethod"></a>bitsadmin gethttpmethod

获取要使用的 HTTP 谓词。

## <a name="syntax"></a>语法

```
bitsadmin /GetHttpMethod <Job>
```

## <a name="parameters"></a>Parameters

|参数|描述|
|---------|-----------|
|作业|该作业的显示名称或 GUID|
