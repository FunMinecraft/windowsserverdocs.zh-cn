# <a name="bitsadmin-help"></a>bitsadmin help



显示命令行使用情况。

## <a name="syntax"></a>语法

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>示例

下面的示例检索命令行帮助。
```
C:\>bitsadmin /help
```
