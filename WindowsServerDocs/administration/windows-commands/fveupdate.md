# <a name="fveupdate"></a>fveupdate



FveUpdate 是在内部使用的工具时，将运行安装程序在计算机从 Windows Vista 或 Windows Server 2008 升级到 Windows 7 或 Windows Server 2008 R2。 它会更新到最新版本与 BitLocker 相关联的元数据。 此工具不能独立运行。
