# <a name="filesystems"></a>文件系统



显示具有焦点的卷的当前文件系统的相关信息，并列出格式化卷时支持的文件系统。

## <a name="syntax"></a>语法

```
filesystems
```

## <a name="remarks"></a>备注

-   若要成功执行此操作，必须选择卷。 使用 "**选择音量**" 命令选择卷并将焦点移动到该卷。
