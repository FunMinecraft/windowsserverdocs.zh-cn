# <a name="ftp-trace1"></a>ftp: trace_1



切换数据包跟踪。

## <a name="syntax"></a>语法

```
trace
```

### <a name="parameters"></a>Parameters

无

## <a name="remarks"></a>备注

**跟踪**运行时显示内部 FTP 函数调用系列**ftp**命令。

## <a name="BKMK_Examples"></a>示例

切换启用和禁用跟踪。
```
trace
```
