# <a name="bitsadmin-peercaching-and-help"></a>bitsadmin 对等缓存和帮助



显示对等**缓存**开关的命令行用法。

## <a name="syntax"></a>语法

```
Bitsadmin /PeerCaching /Help 
```

## <a name="BKMK_examples"></a>示例

以下命令显示对等**缓存**开关的命令行帮助。
```
Bitsadmin /PeerCaching /Help
```
