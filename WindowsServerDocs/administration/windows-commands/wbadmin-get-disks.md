# <a name="wbadmin-get-disks"></a>Wbadmin get 磁盘



列出当前在线的本地计算机的内部和外部磁盘。

若要列出已联机，具有此子命令的磁盘，您必须**Backup Operators**组或**管理员**组，或者您必须被委派了适当的权限。 此外，您必须运行**wbadmin**从提升的命令提示符。 (若要打开提升的命令提示符右键单击**命令提示符**，然后单击**以管理员身份运行**。)

## <a name="syntax"></a>语法

```
wbadmin get disks
```

## <a name="parameters"></a>Parameters

此子命令没有任何参数。
