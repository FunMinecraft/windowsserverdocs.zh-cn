# <a name="nslookup-set-port"></a>nslookup set port



默认 TCP/UDP 域名系统 (DNS) 名称服务器端口更改为指定的值。

## <a name="syntax"></a>语法

```
set port=<Port>
```

## <a name="parameters"></a>Parameters

| 参数 |                                          描述                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<端口 >  | 指定默认 TCP/UDP DNS 名称服务器端口的新值。 默认端口为 53。 |
|   {帮助   |                                              ?}                                               |
