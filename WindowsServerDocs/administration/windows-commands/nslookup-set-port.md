# <a name="nslookup-set-port"></a>nslookup set port



将默认的 TCP/UDP 域名系统（DNS）名称服务器端口更改为指定值。

## <a name="syntax"></a>语法

```
set port=<Port>
```

## <a name="parameters"></a>Parameters

| 参数 |                                          描述                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<Port >  | 为默认的 TCP/UDP DNS 名称服务器端口指定新值。 默认端口为53。 |
|   {帮助   |                                              ?}                                               |
