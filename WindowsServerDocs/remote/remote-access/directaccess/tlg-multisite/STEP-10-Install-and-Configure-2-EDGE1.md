# <a name="step-10-install-and-configure-2-edge1"></a>步骤 10 安装和配置 2 EDGE1

>适用于：Windows 服务器 （半年频道），Windows Server 2016

2 EDGE1 配置由以下内容组成：  
  
- 2 EDGE1 上安装操作系统。 2 EDGE1 上安装 Windows Server 2016 中，Windows Server 2012 R2 或 Windows Server 2012。  
  
- 配置 TCP/IP 属性。 使用这两个网络接口上的静态地址配置 2 EDGE1。  
  
- 配置子网之间的路由。 若要启用公司网络和 2 Corpnet 子网之间的通信，必须配置路由。  
  
- 将 2 EDGE1 加入到 CORP2 域。 将 2 EDGE1 加入到 corp2.corp.contoso.com 域。  
  
- 获取 2 EDGE1 上的证书。 需要 IPsec 连接的 DirectAccess 客户端和远程访问服务器之间和进行身份验证 IP HTTPS 侦听器，当客户端通过 HTTPS 连接证书。  
  
- 向 CORP\User1 提供访问权限。 使用 CORP\User1 用户是远程访问管理员。 若要启用此用户从 EDGE1 对 2 EDGE1 进行更改，必须向用户授予访问权限。  
  
- 在 2 EDGE1 上安装远程访问角色。 若要启用多站点部署，必须在 2 EDGE1 上安装远程访问角色。  
  
2 EDGE1 必须安装两个网络适配器。  
  
## <a name="installOS"></a>2 EDGE1 上安装操作系统  
  
1.  启动 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 的安装。  
  
2.  按照说明完成安装过程中，指定 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 （完全安装） 和本地管理员帐户的强密码。 使用本地管理员账户登录。  
  
3.  连接到具有 Internet 访问权限的网络的 2 EDGE1 和运行 Windows 更新，以便为 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012，安装最新的更新，然后从 Internet 断开连接。  
  
4.  连接到 2 Corpnet 子网，模拟 Internet 到另一个网络适配器。  
  
## <a name="tcpip"></a>配置 TCP/IP 属性  
  
1.  在服务器管理器控制台中，单击**本地服务器**，然后在**属性**区域中，为下的一步**有线以太网连接**，单击的链接。  
  
2.  在中**网络连接**，右键单击连接到 2 Corpnet 子网的网络连接，单击**重命名**，类型**2 Corpnet**，然后按 ENTER。  
  
3.  右键单击**2 Corpnet**，然后单击**属性**。  
  
4.  单击 **“Internet 协议版本 4 (TCP/IPv4)”**，然后单击 **“属性”**。  
  
5.  单击 **“使用下面的 IP 地址”**。 在中**IP 地址**，类型**10.2.0.20**，在**子网掩码**，类型**255.255.255.0**。  
  
6.  单击 **“使用下面的 DNS 服务器地址”**。 在中**首选 DNS 服务器**，类型**10.2.0.1**，然后在**备用 DNS 服务器**，类型**10.0.0.1**。  
  
7.  单击 **“高级”**，然后单击 **“DNS”** 选项卡。  
  
8.  在中**此连接的 DNS 后缀**，类型**corp2.corp.contoso.com**，然后单击**确定**两次。  
  
9. 单击“Internet 协议版本 6 (TCP/IPv6)” ，然后单击“属性” 。  
  
10. 单击**使用下列 IPv6 地址**。 在中**IPv6 地址**，类型**2001:db8:2::20**，在**子网前缀长度**，类型**64**。 单击**使用以下 DNS 服务器地址**，然后在**首选 DNS 服务器**，类型**2001:db8:2::1**，在**备用 DNS 服务器**，类型**2001:db8:1::1**。  
  
11. 单击 **“高级”**，然后单击 **“DNS”** 选项卡。  
  
12. 在中**此连接的 DNS 后缀**，类型**corp2.corp.contoso.com**，然后单击**确定**两次。  
  
13. 上**2 公司网络属性**对话框中，单击**关闭**。  
  
14. 在中**网络连接**窗口中，右键单击连接到 Internet 子网的网络连接，单击**重命名**，类型**Internet**，然后按 enter 键.  
  
15. 右键单击 **“Internet”**，然后单击 **“属性”**。  
  
16. 单击 **“Internet 协议版本 4 (TCP/IPv4)”**，然后单击 **“属性”**。  
  
17. 单击 **“使用下面的 IP 地址”**。 在中**IP 地址**，类型**131.107.0.20**。 在 **“子网掩码”** 框中，键入 **255.255.255.0**。  
  
18. 单击“高级”。 在 **“IP 设置”** 选项卡的 **“IP 地址”** 区域中，单击 **“添加”**。 上**TCP/IP 地址**对话框中**IP 地址**类型**131.107.0.21**，在**子网掩码**类型**255.255.255.0**，然后单击**添加**。  
  
19. 单击 **DNS** 选项卡。  
  
20. 在中**此连接的 DNS 后缀**，类型**isp.example.com**，单击**确定**两次，然后单击**关闭**。  
  
21. 关闭 **“网络连接”** 窗口。  
  
## <a name="routing"></a>配置子网之间路由  
  
1.  上**启动**屏幕上，键入**cmd.exe**，然后按 ENTER。  
  
2.  在命令提示符窗口中，输入以下命令。 输入每个命令后, 按 ENTER。  
  
    ```  
    netsh interface IPv4 add route 10.0.0.0/24 2-Corpnet 10.2.0.254  
    netsh interface IPv6 add route 2001:db8:1::/64 2-Corpnet 2001:db8:2::fe  
    ```  
  
3.  若要检查 2 EDGE1 与 DC1 之间的网络通信，请键入**ping dc1.corp.contoso.com**。  
  
4.  验证是否有四个响应来自任何一个 IPv4 地址 10.0.0.1，或从 IPv6 地址，2001:db8:1::1。  
  
5.  关闭命令提示符窗口。  
  
## <a name="Join"></a>将 2 EDGE1 加入到 CORP2 域  
  
1.  在服务器管理器控制台中，在**本地服务器**，在**属性**区域中，为下的一步**计算机名称**，单击的链接。  
  
2.  在 **“系统属性”** 对话框上的 **“计算机名称”** 选项卡上，单击 **“更改”**。  
  
3.  上**计算机名/域更改**对话框中**计算机名**，类型**2 EDGE1**。 在中**的成员**，单击**域**，类型**corp2.corp.contoso.com**，然后单击**确定**。  
  
4.  如果系统提示您输入用户名和密码，请键入**管理员**其和密码，然后单击**确定**。  
  
5.  当看到一个对话框，欢迎您进入 corp2.corp.contoso.com 域时，请单击**确定**。  
  
6.  当系统提示你必须重新启动计算机时，请单击“确定” 。  
  
7.  在“系统属性”  对话框中单击“关闭” 。  
  
8.  当系统提示你重新启动计算机时，请单击“立即重新启动” 。  
  
9. 重新启动计算机后，单击**切换用户**，然后单击**其他用户**和登录到 CORP2 域管理员帐户。  
  
## <a name="certs"></a>获取 2 EDGE1 上的证书  
  
1.  上**启动**屏幕上，键入**mmc.exe**，然后按 ENTER。  
  
2.  在 MMC 控制台中的“文件”菜单上，单击“添加/删除管理单元”。  
  
3.  在“添加或删除管理单元”对话框上，依次单击“证书”、“添加”、“计算机帐户”、“下一步”、“本地计算机”和“完成”，然后单击“确定”。  
  
4.  在控制台树中的证书管理单元中，打开**证书 （本地计算机） \Personal**。  
  
5.  右键单击**个人**，依次指向**的所有任务**，然后单击**申请新证书**。  
  
6.  单击“下一步”  两次。  
  
7.  上**申请证书**页上，选择**客户端-服务器身份验证**并**Web 服务器**复选框，然后依次**的详细信息注册此证书需要**。  
  
8.  上**证书属性**对话框中，在**主题**选项卡上，在**使用者名称**区域，请在**类型**，选择**公用名**。  
  
9. 在中**值**，类型**2 edge1.contoso.com**，然后单击**添加**。  
  
10. 在“备用名称”区域的“类型”中，选择“DNS”。  
  
11. 在中**值**，输入**2 edge1.contoso.com**，然后单击**添加**。  
  
12. 上**常规**选项卡上，在**友好名称**，类型**IP-HTTPS 证书**。  
  
13. 依次单击“确定”、“注册”和“完成”。  
  
14. 在证书管理单元的详细信息窗格中，验证新证书，其名称 2 edge1.contoso.com 已注册的服务器身份验证预期目的，并注册新证书，其名称 2 edge1.corp2.corp.contoso.com客户端身份验证和服务器身份验证的预期的目的。  
  
15. 关闭控制台窗口。 如果系统提示您保存设置时，请单击**否**。  
  
## <a name="Access"></a>提供对 CORP\User1 访问权限  
  
1.  上**启动**屏幕上，键入**compmgmt.msc**，然后按 ENTER。  
  
2.  在左窗格中，单击**本地用户和组**。  
  
3.  双击**组**，然后双击**管理员**。  
  
4.  上**Administrators 属性**对话框中，单击**添加**，然后在**选择用户、 计算机、 服务帐户或组**对话框中，单击**位置**。  
  
5.  上**位置**对话框中**位置**树中，单击**corp.contoso.com**，然后单击**确定**。  
  
6.  在中**输入要选择的对象名称**类型**User1**，然后单击**确定**。  
  
7.  上**Administrators 属性**对话框中，单击**确定**。  
  
8.  关闭计算机管理窗口。  
  
## <a name="InstallDA"></a>在 2 EDGE1 上安装远程访问角色  
  
1.  在服务器管理器控制台中，在**仪表板**，单击**添加角色和功能**。  
  
2.  单击 **“下一步”** 三次以打开服务器角色选择屏幕。  
  
3.  在“选择服务器角色”对话框中，选择“远程访问”，单击“添加功能”，然后单击“下一步”。  
  
4.  单击“下一步”五次。  
  
5.  在“确认安装选择”  对话框中，单击“安装” 。  
  
6.  在“安装进度”对话框中，验证安装是否成功，然后单击“关闭”。
