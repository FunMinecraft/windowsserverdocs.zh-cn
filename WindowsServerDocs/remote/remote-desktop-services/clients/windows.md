# 要开始使用 Windows 上的远程桌面

>适用于： Windows 10、 Windows 8.1、 Windows Server 2019、 Windows Server 2016、 Windows Server 2012 R2

可以使用 Windows 远程桌面客户端使用 Windows 应用和从另一台 Windows 设备的远程桌面。

使用以下信息以开始。 请务必查看[常见问题解答](remote-desktop-client-faq.md)，如果你有任何问题。

> [!NOTE]
> - 好奇如何在 Windows 客户端的新版本？ 请查看[Windows 上的远程桌面的新增功能？](windows-whatsnew.md)
> - 你可以在任何版本的 Windows 10 上运行客户端。

## 获取 RD 客户端和开始使用它

请按照以下步骤开始使用远程桌面 Windows 10 设备上：

1. 从[Microsoft Store](https://www.microsoft.com/store/p/microsoft-remote-desktop/9wzdncrfj3ps)下载远程桌面客户端。 
2. [设置以接受远程连接的电脑](remote-desktop-allow-access.md)。
3. 添加远程桌面连接或远程资源。 使用连接直接连接到 Windows 电脑和远程资源使用 RemoteApp 程序、 基于会话的桌面或虚拟桌面发布你的管理员。 
4. 因此，你可以快速获得远程桌面固定项。

### 添加远程桌面连接

若要创建远程桌面连接：

1. 在连接中心点击 **+ 添加**，，然后点击**桌面**。
2. 输入你想要连接到计算机的以下信息：
  - **电脑名称**– 计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 你也可以为电脑名称 （例如， **MyDesktop:3389**或**10.0.0.1:3389**） 附加端口信息。
  - **用户帐户**– 要用于访问远程电脑的用户帐户。 点击**+** 添加一个新帐户或选择一个现有帐户。 你可以使用以下格式的用户名：*用户名*、*用户名*或*user_name@domain.com*。 你还可以指定是否选择**每次都询问**提示在连接过程中输入的用户名和密码。
3. 你还可以设置其他选项点击上**显示多个**：
  - **显示名称**– 适用于要连接到电脑的轻松记住名称。 你可以使用任何字符串，但如果未指定一个友好名称，显示 PC 名称。
  - **组**– 指定一组以使其更轻松地查找你连接更高版本。 你可以通过点击添加新的组**+** 或从列表中选择一个。
  - **网关**– 你想要使用连接到虚拟机、 RemoteApp 程序和基于会话的台式机公司内部网络上的远程桌面网关。 从系统管理员获取有关该网关的信息。
  - **连接到管理会话**-使用此选项以连接到控制台会话来管理 Windows server。
  - **交换鼠标按钮**– 使用此选项以鼠标右键按钮交换鼠标左的按钮功能。 （如果这是特别有用远程电脑已配置为左手的用户，但使用右手鼠标。）
  - **我远程会话的分辨率设置为：** -选择你想要在会话中使用的分辨率。 **为我选择**将设置基于客户端的大小的分辨率。
  - **更改的屏幕大小：** – 选择时高静态分辨率会话，你可以选择以使项目在屏幕上的显示较大以提高可读性。 注意： 此设置仅当连接到 Windows 8.1 或更高版本。
  - **更新远程会话上的分辨率调整大小**– 启用时，客户端将动态更新会话分辨率具体取决于客户端的大小。 注意： 此设置仅当连接到 Windows 8.1 或更高版本。
  - **剪贴板**– 启用后，可以复制文本和图像对远程电脑。
  - **音频播放**– 选择该设备要在远程会话期间用于音频。 你可以选择播放声音上的本地设备，远程电脑，或根本。
  - **音频录制**– 启用后，可以使用本地麦克风与远程电脑上的应用程序。
4. 点击**保存**。

需要编辑这些设置？ 点击桌面版名称旁边的溢出菜单 （**...**），然后点击**编辑**。

要删除连接？ 同样，点击溢出菜单 （**...**），然后点击**删除**。

### 添加远程资源
远程资源是 RemoteApp 程序、 基于会话的台式机和发布你的管理员使用远程桌面服务的虚拟机。

若要添加的远程资源：

1. 在连接中心屏幕上，点击 **+ 添加**，，然后点击**远程资源**。 
2. 输入你的管理员提供的**源的 URL**并点击**查找源**。
3. 出现提示时，提供要使用订阅源的凭据。

远程资源将显示在连接中心。


若要删除远程资源：

1. 在连接中心中，点击旁边远程资源溢出菜单 （**...**）。
2. 点击**删除**。

### 固定到开始菜单保存的桌面

若要固定到开始菜单的连接，点击桌面版名称旁边的溢出菜单 （**...**），然后点击**固定到开始菜单**。

现在可以通过点击直接从开始菜单启动远程桌面连接。

## 连接到 RD 网关访问内部资源

远程桌面网关 （RD 网关），可以从任何位置连接到公司网络上的远程计算机在 Internet 上。 你可以创建和管理使用远程桌面客户端在网关。

若要设置的新网关：

1. 在连接中心中，点击**设置**。
2. 旁边网关，点击**+** 若要添加的新网关。 注意： 网关还可以添加在添加新的连接。
3. 输入以下信息：
  - **服务器名称**– 你想要用作网关的计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 你还可以添加端口信息的服务器名称 (例如： **RDGateway:443**或**10.0.0.1:443**)。
  - **用户帐户**-选择或添加用户帐户用于连接到远程桌面网关。 你还可以选择**使用桌面的用户帐户**与用于远程桌面连接使用相同的凭据。
4. 点击**保存**。  

## 全局应用设置

点击**设置**，可以设置以下全局设置你的客户端：

管理项目
- **用户帐户**-使你可以添加、 编辑和删除保存在客户端中的用户帐户。 这是很好地更改后更新帐户的密码。
- **网关**-使你可以添加、 编辑和删除保存在客户端的网关服务器。
- **组**-可以添加、 编辑和删除组保存在客户端。 这些将允许你轻松地组连接。

会话设置
- **开始菜单在全屏模式中的连接**时启用，不论启动连接时，客户端将使用当前的监视器的整个屏幕。
- **开始菜单在新窗口中的每个连接**启用后，每个连接启动在单独窗口中，从而允许你将它们放置在不同的显示器上并使用任务栏它们之间切换。
- **在调整大小时应用：** -允许你控制当客户端窗口调整时，会发生什么情况。 默认为**拉伸的内容，保留纵横比**。
- **使用键盘命令与：** -让我们来指定使用如*WIN*或*ALT + tab 键*的键盘命令的位置。 默认值是仅将它们发送到连接时在完整 scren 会话。
- **阻止屏幕超时**-允许你从超时会话处于活动状态时使屏幕。 当连接对于长时间不需要任何交互时，这非常有用。

应用设置
- **显示桌面预览**-允许你连接到它之前，请参阅连接中心中的桌面设备的预览。 默认情况下，这是设置为**上**。
- **帮助改善远程桌面**-向 Microsoft 发送匿名数据。 我们使用此数据以提高客户端。 若要了解有关我们如何处理此匿名、 专用的数据的详细信息，请参阅[Microsoft 隐私声明](https://privacy.microsoft.com/en-us/privacystatement)。 默认情况下，此设置处于**打开**状态。

### 管理你的用户帐户

连接到桌面或远程资源时，你可以节省要从再次选择的用户帐户。 此外可以在客户端本身，而不是连接到桌面设备时保存用户数据中定义的用户帐户。

若要创建新的用户帐户：

1. 在连接中心中，点击**设置**。
2. 用户帐户旁点击**+** 添加新的用户帐户。
3. 输入以下信息：
   - **用户名**-用户保存与远程连接一起使用的名称。 你可以输入用户名称中的任何以下格式： 用户名，域 \ 用户名，或 user_name@domain.com。
   - **密码**-你指定的用户的密码。 这可以保留为空以在连接期间提示输入密码。
4. 点击**保存**。

若要删除的用户帐户：

1. 在连接中心中，点击**设置**。
2. 选择要从用户帐户列表中删除的帐户。
3. 旁边的用户帐户，请点击编辑图标。
4. 在底部删除用户帐户，请点击**删除此帐户**。
5. 此外可以编辑用户帐户，然后点击**保存**。

## 导航远程桌面会话
启动远程桌面连接时，有工具可用，你可以使用导航会话。

### 开始菜单的远程桌面连接

1. 点击远程桌面连接来启动会话。
2. 如果你尚未已保存的凭据用于连接，系统将提示提供**用户名**和**密码**。
3. 如果您需要验证的远程桌面的证书，查看信息并确保这是你点击**连接**之前信任的电脑。 你还可以选择**不询问有关此证书再次**始终接受此证书。

### 连接栏

连接栏可以访问其他导航控件。 默认情况下，连接栏放置在屏幕顶部的中间。 点击并向左或向右移动它拖动栏。

- **平移控件**的平移控件使屏幕放大，四处移动。 请注意，平移控件仅适用于支持触摸的设备和使用直接触摸模式。
   - 启用 / 禁用平移控件： 点击连接栏中显示的平移控件和缩放屏幕的平移图标。 点击再次来隐藏该控件并返回到其原始分辨率的屏幕连接栏中的平移图标。
   - 使用平移控件的点击和按住平移控件，然后拖动你想要移动屏幕的方向。
   - 移动平移控件的双点击并按住移动在屏幕上的控件的平移控件。
- **其他选项**-点击的其他选项图标以显示会话选择栏和命令栏 （如下所示）。
- **键盘**-触碰键盘图标以显示或隐藏屏幕键盘。 显示键盘时，将自动显示平移控件。

### 命令栏

点击连接栏显示在屏幕的右侧的命令栏上的 **...** 。

- **家庭版**-使用主页按钮以从命令栏返回到连接中心。
  - 或者，你可以为相同的操作使用后退按钮。
  - 活动会话将不会断开连接。
  - 这允许你启动其他连接。
- **断开连接**的使用断开连接按钮终止连接。
  - 只要会话不会终止远程电脑上，你的应用将保持活动状态。
- **全屏**-进入或退出全屏模式。
- **触摸 / 鼠标**-你可以在鼠标模式 （直接触摸和鼠标指针） 之间切换。

### 使用直接在远程会话中触摸手势和鼠标模式

两种鼠标模式都可与会话交互。
- **直接触摸**： 通过所有对会话远程解释该触摸接触。
  - 使用与你应使用触摸屏的 Windows 的方式相同。
- **鼠标指针**： 将本地触摸屏幕转换为较大的触摸板允许在会话中移动鼠标指针。
  - 使用可与触摸板将 Winodws 方式相同。

> [!NOTE]
> 交互与 Windows 8 或更高版本中直接触摸模式支持本机触控笔势。 

| 鼠标模式    | 鼠标操作      | 手势                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接触摸  | 单击鼠标左键           | 1 个手指点击                                                          |
| 直接触摸  | 右键单击          | 1 个手指点击并按住                                                 |
| 鼠标指针 | 单击鼠标左键           | 1 个手指点击                                                          |
| 鼠标指针 | 左键单击并拖动  | 1 个手指双击和保留，然后拖动                               |
| 鼠标指针 | 右键单击          | 2 个手指点击                                                          |
| 鼠标指针 | 右键单击并拖动 | 2 个手指双击和暂停，然后拖动                               |
| 鼠标指针 | 鼠标滚轮          | 2 个手指点击和按住，然后拖动向上或向下                           |
| 鼠标指针 | 缩放                 | 使用 2 个手指并且收缩即可放大或移动手指分开以缩小内容。 |
