# <a name="whats-new-for-the-remote-desktop-client-on-macos"></a>新增功能的 macOS 上的远程桌面客户端？

我们会定期更新[适用于 macOS 的远程桌面客户端](remote-desktop-mac.md)、 添加新功能和修复问题。 请查看以下最新的更新。

如果遇到任何问题，您始终请与我们联系，通过**帮助 > 报告问题**。

## <a name="updates-for-version-1029"></a>针对版本 10.2.9 的更新
*发布的日期：3/6/2019*

- 在此版本中修复了以下远程桌面网关连接问题时将服务器重定向可能发生的放置。
- 我们还解决引起 10.2.8 的 RD 网关回归更新。

## <a name="updates-for-version-1028"></a>针对版本 10.2.8 的更新
*发布的日期：3/1/2019*

- 解决显示时使用的 RD 网关的连接问题。
- 修复了连接时显示的错误的证书警告。
- 解决了某些情况下，其中的菜单栏和停靠会不必要地将启动远程应用程序时其隐藏。
- 重写地址崩溃和挂起的具有已困扰着整个某些用户的剪贴板重定向代码。
- 修复了导致连接中心不必要地滚动启动连接时的 bug。

## <a name="updates-for-version-1027"></a>针对版本 10.2.7 的更新
*发布的日期：2/6/2019*

- 在此版本中，我们将解决图形 mispaints （由服务器编码 bug 而导致） 使用 AVC444 模式时出现。

## <a name="updates-for-version-1026"></a>针对版本 10.2.6 的更新
*发布的日期：1/28/2019*

- 清理应用程序首选项用户界面。
