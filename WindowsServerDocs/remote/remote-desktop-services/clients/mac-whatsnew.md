# <a name="whats-new-for-the-remote-desktop-client-on-macos"></a>什么是新的远程桌面客户端上 macOS？

我们会定期更新[的 macOS 远程桌面客户端](remote-desktop-mac.md)，添加新功能和修复问题。 请查看下面的最新更新。

如果你遇到任何问题，因此你可以始终通过**帮助 > 报告问题**请联系我们。

## <a name="updates-for-version-10210"></a>版本 10.2.10 更新
*发布日期： 3/30/2019年*

- 在此版本中我们解决由新的 macOS 10.14.4 更新导致的不稳定。 我们还修复 mispaints 解码由使用 NVIDIA 硬件的服务器编码的 AVC 编解码器数据时显示。

## <a name="updates-for-version-1029"></a>版本 10.2.9 更新
*发布日期： 3/6/2019年*

- 在此版本中我们固定服务器重定向时可能会发生 RD 网关连接问题将放置。
- 我们还介绍了由 10.2.8 RD 网关回归更新。

## <a name="updates-for-version-1028"></a>版本 10.2.8 更新
*发布日期： 3/1/2019年*

- 解决呈现时使用的 RD 网关的连接问题。
- 连接时显示的固定不正确的证书警告。
- 解决某些情况下，其中的菜单栏和扩展坞将会不必要地在启动远程应用时才能隐藏。
- 返工剪贴板地址崩溃和具有已侵害某些用户的挂起的重定向代码。
- 修复 bug 导致连接中心会不必要地滚动时启动连接。

## <a name="updates-for-version-1027"></a>版本 10.2.7 更新
*发布日期： 2/6/2019年*

- 在此版本中我们解决图形 mispaints （由服务器的编码错误） 使用 AVC444 模式时显示。

## <a name="updates-for-version-1026"></a>版本 10.2.6 更新
*发布日期： 1/28/2019年*

- 添加了对 AVC （420 和 444） 编解码器，可在连接到当前版本的 Windows 10 的支持。
- 在调整为窗口模式，窗口刷新现在发生后立即调整大小以确保内容呈现在正确的内插级别。
- 固定布局 bug 导致订阅源的标头为某些用户重叠。
- 清除应用程序首选项 UI。
- 完美添加/编辑桌面 UI。
- 做了许多适合和完成台式机和源的连接中心磁贴和列表视图的调整。

>存在一个 bug （~/Library 文件夹内深入嵌套） 的 macOS 10.14.0 和 10.14.1 可能会导致".com.microsoft.rdc.application-data_SUPPORT/_EXTERNAL_DATA"文件夹中占用大量磁盘空间。 若要解决此问题，请删除文件夹的内容并升级到 macOS 10.14.2。 请注意，副作用删除文件夹内容的快照图像分配给书签将被删除。 重新连接到远程电脑时，将重新生成这些图像。
