# 在 iOS 上开始使用远程桌面

>适用于： Windows 10、 Windows 8.1、 Windows Server 2019、 Windows Server 2016、 Windows Server 2012 R2

可以使用适用于 iOS 的远程桌面客户端使用 Windows 应用、 资源和台式机从 iOS 设备 （Iphone 和 iPads）。

使用以下信息以开始。 请务必查看[常见问题解答](remote-desktop-client-faq.md)，如果你有任何问题。

> [!NOTE]
> - 好奇 iOS 客户端的新版本？ 请查看[什么是新的远程桌面在 iOS 上？](ios-whatsnew.md)
> - IOS 客户端支持运行 iOS 设备 6.x 和更高版本。

## 获取远程桌面客户端和开始使用它

### 从 iOS 应用商店下载的远程桌面客户端
请按照以下步骤开始使用远程桌面 iOS 设备上：

1. 从[iTunes](https://itunes.apple.com/us/app/microsoft-remote-desktop/id714464092?mt=8)下载 Microsoft 远程桌面客户端。
2. [设置以接受远程连接的电脑](remote-desktop-client-faq.md#how-do-i-set-up-a-pc-for-remote-desktop)。
3. 添加[远程桌面连接](#add-a-remote-desktop-connection)或[远程资源](#add-a-remote-resource)。 使用连接来连接到直接在 Windows 电脑和远程资源使用 RemoteApp 程序，基于会话的桌面或虚拟桌面发布到内部使用 RemoteApp 和桌面连接。 此功能在企业环境中通常可用。

### 下载远程桌面 iOS beta 版客户端
在 iOS 设备，请按照[以下说明](https://aka.ms/rdiosbeta)下载远程桌面 iOS beta 版客户端。

### 添加远程桌面连接

若要创建远程桌面连接： 
1. 在连接中心点击**+**，然后点击**添加 PC 或服务器**。
2. 输入远程桌面连接有关的以下信息：
  - **电脑名称**– 计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 你也可以为电脑名称 （例如， **MyDesktop:3389**或**10.0.0.1:3389**） 附加端口信息。
  - **用户名**– 要用于访问远程电脑的用户名称。 你可以使用以下格式：*用户名*、*用户名*或*user_name@domain.com*。 你还可以指定是否提示输入用户名和密码。
3. 你还可以设置以下附加选项：
  - **友好名称 （可选）** – 适用于要连接到电脑的轻松记住名称。 你可以使用任何字符串，但如果未指定一个友好名称，显示 PC 名称。
  - **网关 （可选）** -你想要使用连接到虚拟机、 RemoteApp 程序和基于会话的台式机公司内部网络上的远程桌面网关。 从系统管理员获取有关该网关的信息。
  - **声音**– 选择该设备要在远程会话期间用于音频。 你可以选择播放声音的本地设备，远程设备上或根本。
  - 它改为**交换鼠标按钮**– 每当鼠标手势会使用发送一个命令鼠标左键时，发送与鼠标右键按钮相同的命令。 这是必要的远程电脑配置为左手鼠标模式。
  - **管理员模式**-连接到运行 Windows Server 2003 或更高版本的服务器上管理会话。
4. 点击**保存**。

需要编辑这些设置？ 按住你想要编辑的桌面，然后点击设置图标。 

### 添加远程资源
远程资源是 RemoteApp 程序、 基于会话的台式机和发布 RemoteApp 和桌面连接使用的虚拟机。

- URL 显示链接到 RD Web 访问服务器，使你能够访问 RemoteApp 和桌面连接。
- 列出配置 RemoteApp 和桌面连接。

若要添加的远程资源：

1. 在连接中心屏幕上，点击**+**，然后点击**添加远程资源**。 
2. 输入远程资源的信息：
   - **源的 URL** -RD Web 访问服务器的 URL。 这将告知客户端搜索与你的电子邮件地址关联的 RD Web 访问服务器还可以在此字段 – 中输入你的公司电子邮件帐户。
   - **用户名**-要用于要连接到的 RD Web 访问服务器的用户名称。
   - **密码**-RD Web 访问服务器要连接到使用的密码。
3. 点击**保存**。

远程资源将显示在连接中心。


## 连接到 RD 网关访问内部资源

远程桌面网关 （RD 网关），可以从任何位置连接到公司网络上的远程计算机在 Internet 上。 你可以创建和管理使用远程桌面客户端在网关。

若要设置的新网关：

1. 在连接中心中，点击**设置 > 网关**。 
2. 点击**添加远程桌面网关**。
3. 输入以下信息：
  - **服务器名称**– 你想要用作网关的计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 你还可以添加端口信息的服务器名称 (例如： **RDGateway:443**或**10.0.0.1:443**)。
  - **用户名**-用户名和密码以用于要连接到远程桌面网关。 你还可以选择**使用连接凭据**与用于远程桌面连接使用的相同的用户名和密码。


## 管理你的用户帐户 

连接到桌面或远程资源时，你可以节省要从再次选择的用户帐户。 你可以通过使用远程桌面客户端管理用户帐户。

若要创建新的用户帐户：

1. 在连接中心中，点击**设置**，，然后点击**用户名**。
2. 点击**添加用户帐户**。
3. 输入以下信息：
   - **用户名**-用户保存与远程连接一起使用的名称。 你可以输入用户名称中的任何以下格式： 用户名，域 \ 用户名，或 user_name@domain.com。
   - **密码**-你指定的用户的密码。 你想要保存用于远程连接每个用户帐户需要有与之相关的密码。
4. 点击**保存**，，然后点击**设置**。
5. 点击**完成**以保存新的配置。

若要删除的用户帐户：

1. 在连接中心中，点击**设置 > 用户名**。
2. 轻扫从右到左若要选择的用户的行。
3. 点击**删除**。



## 导航远程桌面会话
远程桌面会话启动时，有工具可用，你可以使用导航会话。

### 开始菜单的远程桌面连接

1. 点击开始菜单的远程桌面会话对远程桌面连接。 
2. 如果您需要验证的远程桌面的证书，请点击**接受**。 你可以选择始终接受通过滑动**不询问我再次为连接到此计算机**切换到**等**。 

### 连接栏

连接栏可以访问其他导航控件。 

- **平移控件**： 平移控件使屏幕放大，四处移动。 请注意，平移控件仅可使用直接触摸。
   - 启用 / 禁用平移控件： 点击连接栏中显示的平移控件和缩放屏幕的平移图标。 点击再次来隐藏该控件并返回到其原始分辨率的屏幕连接栏中的平移图标。
   - 使用平移控件： 点击并按住平移控件，然后在你想要移动屏幕的方向拖动。
   - 移动平移控件： 双击点击并按住移动在屏幕上的控件的平移控件。
- **连接名称**： 显示当前连接名称。 点击要显示的会话选择栏的连接名称。
- **键盘**： 单击键盘图标，以显示或隐藏键盘。 显示键盘时，将自动显示平移控件。
- **移动连接栏**： 点击和按住连接栏，然后拖放和拖放到屏幕顶部的新位置。

### 会话选择
你可以有多个连接一次打开到不同的电脑。 点击连接栏显示在屏幕左侧的会话选择栏。 会话选择栏可以查看打开连接以及它们之间切换。 

- 打开远程资源会话中的应用之间切换。

    当你连接到远程资源时，你可以通过点击扩展菜单中，从可用的项的列表中选择该会话中打开应用程序之间进行切换。
- 启动新的会话

  你可以开始新应用程序或从桌面会话当前的连接内： 点击**新开始菜单**，然后选择从可用的项的列表。

- 会话的断开连接

  若要断开连接会话点击 X 会话磁贴的左侧。

### 命令栏

命令栏替换为栏开始 8.0.1 版中的实用程序。 你可以在鼠标模式之间切换，并从命令栏返回到连接中心。

## 在远程会话中使用触控笔势和鼠标模式

客户端使用标准触控笔势。 你还可以使用触控笔势复制在远程桌面的鼠标操作。 可用的鼠标模式下表中定义。

> [!NOTE]
> 交互与 Windows 8 或更高版本中直接触摸模式支持本机触控笔势。 有关详细信息 Windows 8 手势，请参阅[触摸： 轻扫、 点击，及更高](https://windows.microsoft.com/en-US/windows-8/touch-swipe-tap-beyond)。

| 鼠标模式    | 鼠标操作      | 手势                                                    |
|---------------|----------------------|------------------------------------------------------------|
| 直接触摸  | 单击鼠标左键           | 1 个手指点击                                               |
| 直接触摸  | 右键单击          | 1 个手指点击并按住                                      |
| 鼠标指针 | 单击鼠标左键           | 1 个手指点击                                               |
| 鼠标指针 | 左键单击并拖动  | 1 个手指双击和保留，然后拖动                    |
| 鼠标指针 | 右键单击          | 2 个手指点击                                               |
| 鼠标指针 | 右键单击并拖动 | 2 个手指双击和暂停，然后拖动                    |
| 鼠标指针 | 鼠标滚轮          | 2 个手指点击和按住，然后拖动向上或向下                |
| 鼠标指针 | 缩放                 | 收缩 2 个手指进行放大或传播 2 个手指以缩小内容 |

## 在远程会话中使用键盘

你可以使用屏幕键盘或远程会话中的物理键盘。

有关屏幕键盘，该按钮用于在键盘上方栏的右侧标准和其他键盘之间切换。

如果你的 iOS 设备启用蓝牙，则客户端将自动检测蓝牙键盘。

请注意，由于在操作系统上的限制，例如 Ctrl、 选项和函数的特殊键将无法按预期方式使用蓝牙键盘。 以下项工作：

- 字母数字键
- 光标键
- 选项卡： 选项卡上的工作原理，但 Shift + tab 键不起作用
- 家庭版 / Pos1: Alt + 向左键 = 家庭版
- 结束： Alt + 向右键 = 结束
- Page Up: Alt + 向上 = Page Up
- Page Down: Alt + 向下 = Page Down
- 选择所有： 命令 + A = Ctrl + A （在大多数程序中的全选）
- 剪切： 命令 + X = Ctrl + X （在大多数程序剪切）
- 复制： 命令 + C = Ctrl + C （在大多数程序复制）
- 粘贴： 命令 + V = Ctrl + V （在大多数程序粘贴）
- 符号： Alt + 字母数字键将会生成不同的符号，具体取决于配置的语言
