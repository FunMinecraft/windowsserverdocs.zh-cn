# <a name="get-started-with-remote-desktop-on-android"></a>开始使用 Android 上的远程桌面

>适用于：Windows 10 中，Windows 8.1、 Windows Server 2019、 Windows Server 2016 中，Windows Server 2012 R2

可以使用适用于 Android 的远程桌面客户端能够直接从你的 Android 设备使用 Windows 应用和桌面。

使用以下信息来开始。 请务必查看[常见问题解答](remote-desktop-client-faq.md)如果有任何问题。

> [!NOTE]
> - Android 客户端的新版本感到好奇？ 请查看[What's new for Android 上的远程桌面？](android-whatsnew.md)
> 可以运行客户端，以及与安装 ChromeOS 53 Chromebook Android 4.1 和更高版本的设备上。 了解有关在 Chrome 上的 Android 应用程序的详细信息[此处](https://sites.google.com/a/chromium.org/dev/chromium-os/chrome-os-systems-supporting-android-apps)。

## <a name="get-the-rd-client-and-start-using-it"></a>获取远程桌面客户端并开始使用它

请执行以下步骤，若要开始使用远程桌面 Android 设备上：

1. 下载中的远程桌面客户端[Google Play](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android)。 
2. [将您的 PC 设置为接受远程连接](remote-desktop-allow-access.md)。
3. 添加远程桌面连接或远程资源。 使用连接来直接连接到 Windows PC 和远程资源，若要使用的 RemoteApp 程序、 基于会话的桌面或虚拟桌面发布在本地。 
4. 创建小组件，以便可以快速获取远程桌面连接。

> [!NOTE]
> 如果你想要前面航班的新功能建议下载我们[Microsoft 远程桌面 Beta](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android.beta)从 Google Play 应用商店应用程序。 

### <a name="add-a-remote-desktop-connection"></a>添加远程桌面连接

若要创建的远程桌面连接：

1. 在连接中心点击 **+** ，然后点击**桌面**。
2. 输入你想要连接到计算机的以下信息：
   - **电脑名称**– 计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 您还可以向 PC 名称追加端口信息 (例如， **MyDesktop:3389**或**10.0.0.1:3389**)。
   - **用户名称**– 要访问远程 PC 使用的用户名。 您可以使用以下格式： *user_name*， *domain\user_name*，或<em>user_name@domain.com</em>。 此外可以指定是否提示输入用户名和密码。
3. 此外可以设置以下附加选项：
   - **友好名称**– 适用于要连接到 PC 易于记忆的名称。 可以使用任意字符串，但如果不指定友好名称，PC 名称将显示。
   - **网关**– 你想要用于连接到虚拟机、 RemoteApp 程序和内部企业网络上的基于会话的桌面的远程桌面网关。 从您的系统管理员获取有关网关的信息。
    需要配置远程桌面网关？
   - **声音**– 选择要在远程会话期间用于音频设备。 您可以选择播放声音或完全不响应本地设备，远程设备上。
   - **自定义显示分辨率**-通过启用此设置设置为连接的自定义分辨率。 解决方法关闭应用时的应用程序的全局设置中定义。
   - **交换鼠标按钮**– 使用此选项可交换的鼠标右键按钮的鼠标按钮功能。 （如果这是特别有用远程 PC 配置为惯用左手的用户，但使用右手鼠标。）
   - **连接到管理会话**-使用此选项可连接到要管理的 Windows 服务器的控制台会话。
   - **将重定向到本地存储**– 在远程 PC 上作为远程文件系统装载本地存储。
4. 点击**保存**。

需要编辑这些设置？ 点击溢出菜单 ( **...** ) 的桌面版，然后点击名称旁边**编辑**。

想要删除的连接？ 同样，点击溢出菜单 ( **...** )，然后点击**删除**。

>[!TIP]
> 如果你获取有关错误 0xf07 （"我们无法连接到远程电脑因为与用户帐户关联的密码已过期"） 的密码不正确，更改你的密码，然后重试。

### <a name="add-a-remote-resource"></a>添加远程资源
远程资源是 RemoteApp 程序、 基于会话的桌面和使用 RemoteApp 和桌面连接发布的虚拟桌面。

若要添加的远程资源：

1. 在连接中心屏幕上，点击 **+** ，然后点击**远程资源源**。 
2. 输入远程资源的信息：
   - **电子邮件或 URL** -RD Web 访问服务器的 URL。 这会告知客户端要搜索与你的电子邮件地址关联的 RD Web 访问服务器还可以在此字段 – 中输入你的公司电子邮件帐户。
   - **用户名称**-要用于连接到 RD Web 访问服务器的用户名。
   - **密码**-要用于连接到 RD Web 访问服务器的密码。
3. 点击**保存**。

将在连接中心中显示的远程资源。


若要删除远程资源：

1. 在连接中心，点击溢出菜单 ( **...** ) 旁边的远程资源。
2. 点击**删除**。
3. 确认删除。

### <a name="widgets--pin-a-saved-desktop-to-your-home-screen"></a>小组件 – Pin 桌面已保存到你的主屏幕

远程桌面应用程序使用 Android 小组件功能支持固定连接到你的主屏幕。 添加小组件的方式取决于所使用的 Android 设备和其操作系统的类型。 下面是添加小组件的最常见方法： 

1. 点击**应用**以启动应用程序菜单。
2. 点击**小组件**。
3. 往下轻扫通过小组件，并查找具有说明"Pin 为远程桌面。"的远程桌面图标
4. 点击按住该远程桌面小组件，将其移动到主屏幕。
5. 当释放该图标时，你将看到已保存远程桌面。 选择你想要将保存到你的主屏幕的连接。

现在可以点击它直接从你的主屏幕中启动远程桌面连接。

> [!NOTE]
> 如果重命名远程桌面应用程序中的桌面连接，不会更新此固定的远程桌面的标签。

## <a name="connect-to-an-rd-gateway-to-access-internal-assets"></a>连接到 RD 网关访问内部资源

远程桌面网关 （RD 网关），可以从任何位置连接到公司网络上的远程计算机在 Internet 上。 可以创建和管理网关使用远程桌面客户端。

若要设置新的网关：

1. 在连接中心，点击**设置 > 网关**。 点击 **+** 添加新的网关。
2. 输入以下信息：
   - **服务器名称**– 你想要用作网关的计算机的名称。 这可以是 Windows 计算机名称、 Internet 域名或 IP 地址。 此外可以添加端口信息的服务器名称 (例如：**RDGateway:443**或**10.0.0.1:443**)。
   - **用户名称**-用户名和密码以用于连接到远程桌面网关。 您还可以选择**使用桌面用户帐户**若要使用相同的凭据与用于远程桌面连接。

## <a name="manage-your-user-accounts"></a>管理用户帐户

当连接到桌面或远程资源时，您可以保存要再次通过选择的用户帐户。 此外可以在客户端本身，而不是连接到桌面时保存用户数据中定义的用户帐户。

若要创建新的用户帐户：

1. 在连接中心，点击**设置**，然后点击**用户帐户**。
2. 点击 **+** 添加新的用户帐户。
3. 输入以下信息：
   - **用户名称**-要保存以用于远程连接的用户的名称。 可以在任何以下格式输入用户名： 域 \ 用户名，user_name 或user_name@domain.com。
   - **密码**-您指定的用户的密码。 要用于保存要用于远程连接每个用户帐户需要具有与之关联的密码。
4. 点击**保存**。

若要删除的用户帐户：

1. 在连接中心，点击**设置 > 用户帐户**。
2. 点击并按住以将其选中列表中的用户帐户。 您可以选择多个用户。
3. 点击垃圾桶可将其删除所选的用户。

## <a name="navigate-the-remote-desktop-session"></a>导航远程桌面会话
启动远程桌面连接时，有工具可用于导航会话。

### <a name="start-a-remote-desktop-connection"></a>启动远程桌面连接

1. 点击以启动会话的远程桌面连接。 
2. 如果您要求验证远程桌面的证书，请点击**Connect**。 您还可以选择**不要再询问我是否连接到此计算机**始终接受的证书。

### <a name="manage-global-app-settings"></a>管理全局应用程序设置

Android 客户端中，可以设置以下全局设置：

- **显示桌面预览**-允许你连接到它之前，请参阅 desktop 中连接中心的预览。 默认情况下，此设置为**上**。
- **捏合缩放到**-使您可以使用捏合缩放手势。 如果要通过远程桌面使用的应用支持多点触控 （在 Windows 8 中引入），打开此设置**关闭**。
- **帮助以提高远程桌面**-向 Microsoft 发送匿名数据。 我们使用此数据来改进客户端。 您可以了解有关我们如何处理此匿名的私有数据的详细信息，请参阅[远程桌面客户端 Privacy Statement](https://www.microsoft.com/privacystatement/RemoteApp/Default.aspx)。 默认情况下，此设置是**上**。
- **显示**-有两个显示器的全局设置：
  - **方向**-设置你的会话的首选的方向 （横向或纵向）。 
    >[!NOTE]
    > 如果您连接到运行 Windows 8 或 Windows 的较旧版本的电脑，该会话不会正确缩放。 最好的方法就是从 PC，断开连接，然后重新连接在你想要使用的方向。 更好的选择是至少升级到 PC Windows 8.1。

  - **解析**-设置要用于全局桌面连接所需的分辨率。 如果你已设置的单个应用程序或连接自定义解析，将不会更改此设置的。
    >[!NOTE]
    >当你更改其中一个显示设置时，它们只适用于新的连接从该点上。 若要查看已连接断开的连接，并再次连接的会话中的更改。

### <a name="connection-bar"></a>连接栏

连接栏可以访问其他导航控件。 默认情况下，连接栏位于屏幕顶部的中间。 双点击和拖动竖线向左或向右移动它。

- **平移控件**:平移控件启用屏幕放大和移动。 请注意平移控件才可以使用直接触摸。
   - 启用/禁用平移控件：点击连接栏显示平移控件和缩放屏幕中的平铺图标。 点击连接栏，再次以隐藏控件并返回到其原始分辨率的屏幕中平铺图标。
   - 使用平移控件：点击按住平移控件，然后你想要移动屏幕的方向拖动。
   - 移动平移控件：双点击并按住移动屏幕上的控件的平移控件。
- **其他选项**:点击其他选项图标以显示会话选择栏和命令栏 （见下文）。
- **键盘**:点击键盘图标以显示或隐藏键盘。 显示键盘时，将自动显示平移控件。
- **移动连接栏**:点击并按住连接栏，然后拖放到屏幕顶部的新位置。


### <a name="command-bar"></a>命令栏

点击连接栏，以在屏幕右侧显示的命令栏。 您可以切换鼠标模式 （直接触摸和鼠标指针）。 使用主页按钮返回到连接中心从命令栏。 或者你可以使用后退按钮相同的操作。 活动会话将不被中断。 


### <a name="use-direct-touch-gestures-and-mouse-modes-in-a-remote-session"></a>使用直接在远程会话中触摸手势和鼠标模式

客户端使用标准的触摸手势。 触摸手势还可用于复制在远程桌面上的鼠标操作。 下表中定义可用的鼠标模式。

> [!NOTE]
> 交互与 Windows 8 或更高版本中直接触摸模式支持本机触摸手势。 

| 鼠标模式    | 鼠标操作      | 手势                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接触摸  | 单击鼠标左键           | 1 指点击                                                          |
| 直接触摸  | 右键单击          | 1 指点击并按住                                                 |
| 鼠标指针 | Zoom                 | 使用 2 个手指，并使用捏合手势进行放大或移动手指分开以缩小。 |
| 鼠标指针 | 单击鼠标左键           | 1 指点击                                                          |
| 鼠标指针 | 左键单击并拖动  | 1 手指双点击并按住，然后拖动                               |
| 鼠标指针 | 右键单击          | 2 个手指点击                                                          |
| 鼠标指针 | 右键单击并拖动 | 2 个手指双点击并按住，然后拖动                               |
| 鼠标指针 | 鼠标滚轮          | 2 个手指点击并按住，然后向上或向下拖动                           |
