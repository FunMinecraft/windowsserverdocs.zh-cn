# <a name="install-updates"></a>安装更新
我们建议您安装更新，如果可用。 安装更新需要 Internet 连接。  

1.  从“开始”  屏幕打开“控制面板” 。  
  
2.  在控制面板中，键入**更新**，然后单击**检查更新**。  
  
3.  如果在 Windows Update 网站列出了您的计算机所需的任何更新，安装更新。
