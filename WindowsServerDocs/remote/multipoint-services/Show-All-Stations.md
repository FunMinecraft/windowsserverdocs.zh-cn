# <a name="show-all-stations"></a>显示所有工作站
作为 MultiPoint 服务系统上的管理员，你可能想要查看所有已连接到您的系统，包括非工作站\-活动或已注销的工作站。 例如，每个许可协议，你可能需要检查已连接的工作站数量的 MultiPoint 服务系统。 您可以查看此信息使用 MultiPoint 管理器。  
  
1.  在常规模式下，单击“桌面”选项卡。  
  
2.  在“常规任务”下，单击“显示所有工作站”。
