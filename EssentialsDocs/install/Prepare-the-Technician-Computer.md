# <a name="prepare-the-technician-computer"></a>准备技术人员计算机

>适用于：Windows Server 2016 Essentials，Windows Server 2012 R2 Essentials 中，Windows Server 2012 Essentials

技术人员计算机包含的工具和文档中的 Windows 评估和部署工具包 (ADK)，并且你可以自定义你打算部署的操作系统的图像。 完成以下步骤来准备技术人员计算机。  
  
-   准备有 Windows 7、Windows 8 或 Windows 10 的计算机。  
  
-   在运行 Windows 7、Windows 8 或 Windows 10 中通过的计算机上安装 Windows ADK [Microsoft 网站](https://go.microsoft.com/fwlink/?LinkID=248647)。
