# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>使用连接到 Windows Server Essentials 我服务器应用

>适用于：Windows Server 2016 Essentials，Windows Server 2012 R2 Essentials 中，Windows Server 2012 Essentials

Windows Server Essentials 我服务器应用允许你连接到的资源并执行你从你的笔记本电脑、 PC 或 Surface 设备的 Windows Server Essentials 服务器上的光管理任务。 我服务器，在你可以管理用户、 设备和通知，并使用服务器上的共享文件。 当你处于离线状态时，你可以继续使用我的服务器; 在最近访问的文件离线状态下更改会自动与服务器同步的下次连接。  
  
 如果你服务器运行的 Windows Server Essentials 操作系统，使用原始我服务器应用。 如果你服务器运行的 Windows Server Essentials 操作系统，必须使用更新我的 Server 2012 R2 的应用。 可从这两个应用下载[适用于 Windows 的应用](https://windows.microsoft.com/windows-8/apps)。  
  
> [!TIP]

>  若要从 Windows Phone 中访问 Windows Server Essentials 中的资源，请使用我的服务器手机应用 （适用于 Windows Server Essentials) 或我 Server 2012 R2 手机应用 （适用于 Windows Server Essentials)。 我服务器的手机应用的信息，请参阅[用于 Windows Phone 的我服务器应用](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2)。 我 Server 2012 R2 的手机应用的信息，请参阅博客条目[我 Server 2012 R2 的 Windows 和 Windows Phone 应用](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx)。

##  <a name="BKMK_Install"></a>安装我服务器  
 你可以下载从相应我服务器应用[适用于 Windows 的应用](https://windows.microsoft.com/windows-8/apps)存储。 如果计算机中安装有 Windows 8 或 Windows RT 操作系统，并且你想要连接到服务器上的 intranet 使用服务器的名称，你还需要安装证书，从服务器。  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>若要安装在你基于 Windows 的笔记本电脑、 PC 或 Surface 设备上的服务器我或我 Server 2012 R2  
  
1.  安装来自相应我服务器应用[适用于 Windows 的应用](https://windows.microsoft.com/windows-8/apps)到你基于 Windows 的笔记本电脑、 PC 或 Surface 设备上。  
  
    |服务器操作系统|下载|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[我服务器](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[我 Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  如果你想要将无法使用服务器名称 intranet，通过连接到你的 Windows Server Essentials 服务器，并且你的设备具有 Windows 8 或 Windows RT 操作系统，默认 CAROOT.cer 证书你在计算机上安装通过执行以下步骤。 Windows 8.1 和 Windows RT 8.1 中不需要的证书手动安装。  
  
###  <a name="BKMK_InstallCert"></a>若要在 Windows 8、 Windows 8.1 或 Windows RT 设备上为我服务器安装证书  
  
1.  打开 web 浏览器你在计算机上，并从服务器下载默认证书，CAROOT.cer。 若要执行此操作，键入以下命令，替换服务器的名称 (例如， **marketing.contoso.com**) 为 <*服务器名称*>:  
  
     **http://<servername\>/connect/default.aspx?get=caroot.cer**  
  
2.  安装证书：  
  
    1.  在**开始**页上，打开**文件资源管理器**。  
  
    2.  确保显示隐藏的项目和文件扩展名： 上**视图**选项卡上，在**显示/隐藏**组中，选择**隐藏的项目**和**文件扩展名**复选框。  
  
    3.  导航到包含 CAROOT.cer 下载的文件，你只需的文件夹。  
  
    4.  右键单击 CAROOT.cer 文件，然后单击**打开**。  
  
    5.  在**证书**对话框中，单击**安装证书**。  
  
    6.  选择**本地计算机**作为安装位置，然后单击**下一步**。  
  
    7.  在**证书官方商城**向导页上，选择**放置在以下应用商店中的所有证书**，并使用**浏览**选择**信任根证书颁发机构**存储。 然后单击**完成**。

##  <a name="BKMK_ConnectServer"></a>如何从你的本地网络连接到服务器  
 要成功使用 Windows Server Windows Phone 的 Essentials、 Windows 8 和 Windows 8.1 中的我服务器应用，必须先在你的设备上安装服务器证书。 这将设备连接到你的本地网络中运行 Windows Server Essentials 服务器。 使用下面的过程连接到你在本地网络的服务器，并在你的 Windows Phone 或运行的 Windows 8 或 Windows 8.1 的计算机上安装服务器证书。  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>若要从 Windows Phone 连接到服务器  
  
1.  你运行的 Windows 8 或 Windows 8.1 的 Windows Phone，在打开 Internet Explorer。  
  
2.  在地址栏中，键入**http://<yourservername\>/connect/default.aspx?get=caroot.cer**，然后按 Enter。  
  
3.  当提示你安装 caroot.cer 证书时，请单击**安装**。  
  
4.  等待证书安装完成。  
  
    > [!NOTE]
    >  证书安装完成后，你可以登录到我服务器应用的 Windows Phone 使用服务器名称和网络的凭据。  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>从一台运行 Windows 8 或 Windows 8.1 计算机连接到服务器，在你的本地网络  
  
1.  在计算机上运行的 Windows 8 或 Windows 8.1，打开 Internet Explorer。  
  
2.  在地址栏中，键入**http://<yourservername\>/connect/default.aspx?get=caroot.cer**，然后按 Enter。  
  
3.  当提示你安装 caroot.cer 证书时，请单击**打开**。  
  
4.  在**证书**对话框中，单击**安装证书**。  
  
5.  在证书导入向导中，选择**本地计算机**作为应用商店的位置。  
  
6.  选择**放置在以下应用商店中的所有证书**，然后浏览到**信任根证书颁发机构**位置。  
  
7.  按照说明来完成向导。  
  
    > [!NOTE]
    >  证书安装完成后，你可以登录到的我的服务器应用 Windows 8 或 Windows 8.1 使用服务器名称和网络的凭据。
